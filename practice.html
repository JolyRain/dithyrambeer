Название проекта

Разработка веб-приложения "Агрегатор рецензий на товары"
Одиночный проект

Постановка задачи

Разработать веб-приложение, которое позволяет пользователям оставлять собственные отзывы на товары из различных магазинов.
Функционал приложения
  • Регистрация пользователей
  • Два уровня доступа – пользователь, системный администратор
  • Каждый уровень доступа имеет свою особенную функциональность
  • Хранение данных с использованием СУБД
Функционал приложения со стороны пользователя
  • Возможность писать отзыв на товар с развернутой рецензией или простой оценкой к нему
  • Просмотр отзывов других пользователей
  • Доступ в личный кабинет с информацией о пользователе и списком оставленных пользователем отзывов
Функционал приложения со стороны системного администратора
  • Полный функционал пользователя
  • Возможность добавлять новые товары, удалять уже существующие
  • Возможность добавлять новых пользователей, удалять уже существующих
  • Удаление товаров и отзывов пользователей к ним
Задачи по проекту - Frontend
  • Реализация главной страницы со списком товаров
  • Реализация формы для регистрации пользователей
  • Реализация личного кабинета пользователя и администратора
  • Реализация страницы товара с отзывами пользователей
Задачи по проекту - Backend
  • Реализация базы данных пользователей и товаров
  • Реализация обработки запросов к СУБД со стороны пользователей на добавление отзывов к товарам
  • Реализация обработки запросов к СУБД со стороны системного администратора на добавление/удаление товаров, отзывов к ним и пользователей
